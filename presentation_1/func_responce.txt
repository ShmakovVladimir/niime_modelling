Для построения функции двух переменных с положительно определенной матрицей Гессе и избегания квадратичной формы, можно использовать нелинейную функцию, например, комбинацию экспоненциальных или тригонометрических функций. Вот пример функции, которая удовлетворяет условию:

### Функция:
\[ f(x, y) = e^{x^2 + y^2} + \sin(x + y) + x^2 + y^2 \]

Эта функция не является квадратичной, и за счёт экспоненты и синуса имеет более сложную нелинейную структуру.

### Градиент (аналитическое выражение):
Градиент функции \( f(x, y) \) — это вектор, содержащий частные производные функции по каждой из переменных \(x\) и \(y\).

1. Частная производная по \( x \):
\[
\frac{\partial f}{\partial x} = 2x e^{x^2 + y^2} + \cos(x + y) + 2x
\]

2. Частная производная по \( y \):
\[
\frac{\partial f}{\partial y} = 2y e^{x^2 + y^2} + \cos(x + y) + 2y
\]

Таким образом, градиент:
\[
\nabla f(x, y) = \left( 2x e^{x^2 + y^2} + \cos(x + y) + 2x, 2y e^{x^2 + y^2} + \cos(x + y) + 2y \right)
\]

### Матрица Гессе (аналитическое выражение):
Матрица Гессе — это матрица вторых частных производных функции. Для нашей функции это будет:

1. Вторые производные по \( x \):
\[
\frac{\partial^2 f}{\partial x^2} = 2e^{x^2 + y^2} + 4x^2 e^{x^2 + y^2} - \sin(x + y) + 2
\]

2. Вторая производная по \( y \):
\[
\frac{\partial^2 f}{\partial y^2} = 2e^{x^2 + y^2} + 4y^2 e^{x^2 + y^2} - \sin(x + y) + 2
\]

3. Смешанная производная по \( x \) и \( y \):
\[
\frac{\partial^2 f}{\partial x \partial y} = 4xy e^{x^2 + y^2} - \sin(x + y)
\]

### Итоговая матрица Гессе:
\[
H(f) = 
\begin{pmatrix}
2e^{x^2 + y^2} + 4x^2 e^{x^2 + y^2} - \sin(x + y) + 2 & 4xy e^{x^2 + y^2} - \sin(x + y) \\
4xy e^{x^2 + y^2} - \sin(x + y) & 2e^{x^2 + y^2} + 4y^2 e^{x^2 + y^2} - \sin(x + y) + 2
\end{pmatrix}
\]

### Проверка положительной определённости:
Так как экспоненциальный член \( e^{x^2 + y^2} \) положителен, а синусовые члены \( \sin(x + y) \) имеют ограниченное значение, можно ожидать, что матрица Гессе этой функции будет положительно определённой на большом диапазоне значений \(x\) и \(y\), включая указанный диапазон \((-5, 5)\).
